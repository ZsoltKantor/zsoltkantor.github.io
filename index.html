Yt-dlp BcGUI
Short description:
Yt-dlp BcGUI is a graphical user interface for the command line tool yt-dlp.
The Bc part in the name comes from the word basic, so it is not a fancy GUI, instead it was designed to run fast (was written in pure C language) with a simple interface.
You can do things like download media, get a formatted list of the available media formats, update the yt-dlp downloader etc.
